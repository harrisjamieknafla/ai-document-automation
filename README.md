# ai-document-automation
Simple Python AI tool for automatic text summarization
AI Document Summarizer

This project is a simple Python-based tool that automatically summarizes long text inputs using artificial intelligence.

The goal of the project is to explore how AI and automation can be used to process large amounts of text and extract the most important information efficiently.

Features

- Automatic summarization of long text
- Simple command line interface
- Demonstrates basic AI-powered text processing
- Example of automation using Python

Technologies Used

- Python
- OpenAI API
- Natural Language Processing concepts

How it works

The user enters a long piece of text into the program.  
The system processes the text and generates a shorter summary that captures the most important information.

This demonstrates how artificial intelligence can support document processing and information extraction tasks.

Purpose of the Project

This project was created to explore the potential of artificial intelligence and automation tools in processing and summarizing textual data.

To run this project you need an OpenAI API key. Set your API key as an environment variable: export OPENAI_API_KEY="your-api-key"

It also serves as a small example project demonstrating how modern AI APIs can be integrated into Python applications.
